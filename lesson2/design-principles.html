PRINCIPLES OF DESIGN

  • PARC: Contrast

  • Total Energies
  • totalenergies.com/

This site makes use good use of contrast as it displays text in white color over the blue background, while the text in the darker, grayish background is black, ultimately improving readability.

  • Fitt's Law

  • Ghana Post
  • ghanapost.com.gh/

This site takes advantage Fitt's law because the action items and interactive items are sufficiently large and conspicuous, so it will not take a user much time to locate a desired button or item and select it.

  • PARC: Repitition

  • Code Web Limited
  • codewebltd.com/

Repitition can be observed in this site from the blue design on the top right corner of the various widgets. They are almost identical, with the only difference being the small icon within the blue design. The repetition of this blue design tells the user that he/she is still scrolling through widgets.
