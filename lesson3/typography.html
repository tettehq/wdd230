Typography

The following table describes the font, fontsize, line height, and styles to be used in this site.

OBJECT FONT SIZE COLOR
Default Paragraph Text(p)

 Josefin Sans default #000000
Heading text(h1,h2) Gluten default white(#FFFFFF)
Navigation Anchor Selected Josefin Sans 1.5em white(#FFFFFF)
Navigation Anchor Unselected Josefin Sans 1.5em Sapphire blue(#1768AC)

Navigation

Navigation will be near the top of the page with the following color scheme

  • Unselected link will be Sapphire blue(#1768AC) in white(#FFFFFF) background
  • Link being hovered on will have a reverse color scheme as unselected link
  • Selected link will have denim blue(#2541B2)
